# Data Sources

## Fraud Detection Dataset
- **Title:** Credit Card Fraud Detection
- **Origin:** Kaggle — https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
- **License:** DbCL-1.0 (Database Contents License)
- **Download Location:** `data/raw/creditcard_fraud/`
- **Notes:** European card transactions (anonymised features). Highly imbalanced; use for classification and anomaly detection.
